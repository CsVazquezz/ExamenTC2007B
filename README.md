# Sudoku_TC2007B

- Nombre completo: Carlos Martinez Vazquez  
- Matrícula: A01711730 
- Plataforma: iOS

Brief explanation:
Sudoku is a 9x9 logic puzzle where the goal is to fill every row, column and 3x3 block with numbers 1–9 without repeats. This app fetches puzzles from an API https://www.api-ninjas.com/api/sudoku (handles JSON nulls), provides an offline demo fallback, and supports saving/loading in-progress games to device storage.
